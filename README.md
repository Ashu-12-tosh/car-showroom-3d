# 🚗 Apex GT - 3D Car Showroom Configurator

   Live Demo :-  https://car-showroom-3d-beige.vercel.app

A premium interactive 3D car configurator built using React, TypeScript, Three.js, React Three Fiber, and WebGL.

This project demonstrates real-world 3D web development concepts including vehicle visualization, camera systems, configurable materials, HDRI lighting, reflections, hotspots, performance monitoring, and interactive controls.

---

## ✨ Features

### 3D Vehicle Visualization

* Real-time 3D car rendering
* Physically based materials
* HDRI environment reflections
* Dynamic lighting system
* Real-time shadows

### Interactive Camera System

* Orbit controls
* Smooth camera transitions
* Multiple camera presets

  * Front View
  * Rear View
  * Side View
  * Top View
  * Interior View

### Vehicle Customization

* Paint color configurator
* Material customization
* Wheel configuration system
* Interior customization support

### Interactive Components

* Door controls
* Hood controls
* Trunk controls
* Hotspot information system

### Environment Controls

* Day Mode
* Night Mode
* HDRI environment switching

### Utility Features

* Screenshot capture
* Performance monitoring
* FPS tracking
* Draw call monitoring
* Responsive UI

## 🚀 Getting Started

### Clone Repository

```bash
git clone <repository-url>
cd car-showroom
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
